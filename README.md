## 🚀 Como Publicar Online

### Opção 1: Netlify (Mais Fácil)
1. Acesse [netlify.com](https://netlify.com)
2. Arraste esta pasta inteira para o site
3. Pronto! Você terá uma URL como: `https://seu-convite.netlify.app`

### Opção 2: GitHub Pages
1. Crie um repositório no GitHub
2. Faça upload dos arquivos
3. Ative GitHub Pages nas configurações
4. URL: `https://seu-usuario.github.io/nome-repositorio`

### Opção 3: Vercel
1. Acesse [vercel.com](https://vercel.com)
2. Importe o projeto
3. Deploy automático
4. URL: `https://cha-revelacao.vercel.app`
